# TCP vs UDP

What is the difference between TCP and UDP?

---ANSWER---

TCP is connection-oriented, reliable, and guarantees delivery order. UDP is connectionless, fast, but does not guarantee delivery or order.

### Life Analogy
TCP is a certified mail delivery where you must sign for each package. UDP is throwing newspapers onto driveways from a moving truck.

### Key Points
- TCP: Reliable, slower.
- UDP: Unreliable, faster (video streaming).
