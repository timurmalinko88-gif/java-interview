# Load Testing vs Stress Testing

What is the difference between Load Testing and Stress Testing?

---ANSWER---

**Load Testing:** Evaluates the system's performance under expected normal and peak conditions to ensure it meets SLAs.
**Stress Testing:** Pushes the system beyond normal conditions until it breaks, to see how it handles failure and whether it recovers gracefully.

### Life Analogy
Load testing a bridge is driving heavy trucks over it to see if it holds up. Stress testing is putting tanks on it until it collapses to see which beam snaps first.

### Key Points
- Load: Expected conditions.
- Stress: Breaking point.
