# CORS

What is CORS and why is it used?

---ANSWER---

Cross-Origin Resource Sharing (CORS) is a security feature enforced by browsers that restricts web pages from making requests to a different domain than the one that served the web page. Servers must send specific CORS headers to allow it.

### Life Analogy
Like a bouncer at a club checking if your ID (origin) is on the guest list (allowed origins) before letting you in.

### Key Points
- Browser security feature.
- Prevents malicious scripts.
- Configured on the server.
