# HTTP Methods

What are the most common HTTP methods and their uses?

---ANSWER---

1. GET: Retrieve data.
2. POST: Create new data.
3. PUT: Update/replace existing data.
4. PATCH: Partially update data.
5. DELETE: Remove data.

### Life Analogy
GET is reading a book, POST is writing a new chapter, PUT is replacing a chapter, DELETE is tearing a page out.

### Key Points
- CRUD maps to POST, GET, PUT, DELETE.
- GET should be safe (no side effects).
