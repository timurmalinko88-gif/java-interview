Найдите ошибку в следующем коде и объясните, как её исправить:

```java
public class Counter {
    private int count = 0;

    public void increment() {
        count++;
    }

    public int getCount() {
        return count;
    }
}
```

## Разбор
**Проблема:** 
Операция `count++` не является атомарной. Она состоит из трех скрытых шагов:
1. Чтение текущего значения.
2. Увеличение на 1.
3. Запись нового значения.
Если несколько потоков вызовут `increment()` одновременно, произойдет "состояние гонки" (race condition), и часть инкрементов потеряется.

**Как исправить:**
Для простых счетчиков лучше всего использовать класс `AtomicInteger`:

```java
import java.util.concurrent.atomic.AtomicInteger;

public class Counter {
    private AtomicInteger count = new AtomicInteger(0);

    public void increment() {
        count.incrementAndGet();
    }

    public int getCount() {
        return count.get();
    }
}
```
*Альтернатива:* добавить ключевое слово `synchronized` к методу `increment()`, но это может работать медленнее при большом количестве потоков.
