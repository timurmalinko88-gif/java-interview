В чем разница между абстрактным классом и интерфейсом в Java? 
Перечислите основные отличия.

---ANSWER---

1. **Множественное наследование:** Класс может реализовать (`implements`) несколько интерфейсов, но может наследоваться (`extends`) только от одного абстрактного класса.
2. **Состояние:** Абстрактный класс может иметь обычные поля (состояние). Интерфейс может содержать только константы (`public static final`).
3. **Конструкторы:** В абстрактном классе можно объявить конструктор (для инициализации полей), в интерфейсе конструкторов быть не может.
4. **Модификаторы:** В интерфейсах методы по умолчанию `public` (и с Java 8 `default`/`static`, с Java 9 `private`). В абстрактном классе методы могут быть `protected` и т.д.
