## 🌟 Key Features & Latest Innovations

### 🗺️ Curated Learning Tracks (Roadmaps)
- **Junior Express Track** (50 Questions): Rapid preparation covering Core Java, Collections, and OOP fundamentals.
- **Middle Spring & Microservices Track** (150 Questions): Focused track covering Spring Boot, REST APIs, Microservices architecture, SQL/NoSQL, and Messaging.
- **Senior Architect Track** (100 Questions): Deep dive into System Design, JVM Performance Tuning, High Availability, and Concurrency.
- **AI & Modern Java 21 Track** (40 Questions): Cutting-edge track covering Virtual Threads, Scoped Values, Spring AI, Vector DBs, and LLM integrations.
- **My Roadmap Drawer**: Diagnostic adaptive quiz that analyzes knowledge gaps and constructs a personalized learning track.

### 🎥 Dynamic YouTube Video Explanations
- **One-Click Video Tutorials**: Integrated YouTube search shortcut button on every question card (`btn-youtube`).
- **Targeted Search Queries**: Automatically queries curated Java video tutorials and code walkthroughs for deeper comprehension.

### 🤖 2025-2026 Modern Tech Stack Focus
- **AI, LLM & RAG**: Spring AI, Embeddings, PgVector, Prompt Engineering, Function Calling.
- **Modern Java 21+**: Virtual Threads (Project Loom), Scoped Values, Structured Concurrency, Record Patterns, Pattern Matching for switch.
- **Kafka & Messaging**: Partitioning, Consumer Groups, Exactly-Once Semantics, Idempotency, Dead Letter Queues (DLQ).
- **Live Coding & Refactoring**: Interactive code debugging, bug hunting, and algorithm optimization.
- **Behavioral & STAR Method**: HR questions, conflict resolution, and communication using the STAR methodology.

### 📊 Progress Bars 2.0 & Rank Tracker
- **Dynamic Rank Tracker**: Visual progression monitoring from *Junior Trainee* to *Java Architect*.
- **Topic Micro-Progress**: Individual progress indicators for each category showing percentage of mastered questions.
- **Experience Points (XP)**: Earn XP by reviewing questions, completing SRS memory cards, and conquering mock interviews.

### 🧠 Spaced Repetition System (SRS)
- **Leitner Box Memory System**: Smart review scheduling based on recall accuracy.
- **Adaptive Memory Cards**: Focuses daily practice on weak areas to maximize retention.

### 🎙️ Mock Interview Simulator
- **Customizable Sessions**: Configure mock interviews by topic, difficulty (Junior, Middle, Senior), and question volume (5, 10, 15, 20).
- **Interactive Evaluation**: Real-time scoring (❌ Missed, ⚠️ Partial, ✅ Nailed It) with an end-of-session evaluation Scorecard.

## 🚀 Quick Start & Local Development

No complex setup or build steps required. The application runs natively in any modern web browser.

### 1. Clone the Repository
```bash
git clone https://github.com/timurmalinko88-gif/java-interview.git
cd java-interview
```

### 2. Start a Local HTTP Server
Launch a local server using Python's built-in HTTP module:

```bash
# Python 3
python -m http.server 8000
```
Then navigate to: **[http://localhost:8000](http://localhost:8000)** in your browser.

---

## 🔄 Building / Re-indexing (`build.py`)

If you add new questions or modify Markdown files in `public/questions/`, update `public/index.json` by running the build script:

```bash
python build.py
```

The script automatically:
1. Scans `public/questions/**/*.md` recursively.
2. Parses frontmatter metadata (handles YAML delimited by `---` or key-value headers).
3. Assigns IDs, topics, difficulty levels, estimated times, and formats.
4. Generates a clean, optimized `public/index.json`.
