# HTTP Status Codes

Explain the classes of HTTP status codes.

---ANSWER---

- 1xx: Informational.
- 2xx: Success (e.g., 200 OK, 201 Created).
- 3xx: Redirection (e.g., 301 Moved Permanently).
- 4xx: Client Error (e.g., 400 Bad Request, 404 Not Found).
- 5xx: Server Error (e.g., 500 Internal Server Error).

### Life Analogy
2xx: "Here is your coffee." 4xx: "You ordered something not on the menu." 5xx: "Our espresso machine exploded."

### Key Points
- 4xx = your fault.
- 5xx = my fault.
