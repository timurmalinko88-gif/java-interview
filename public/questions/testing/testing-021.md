# HTTP vs HTTPS

What is the difference between HTTP and HTTPS?

---ANSWER---

HTTPS is HTTP with encryption. It uses TLS (Transport Layer Security) to encrypt requests and responses, ensuring that data cannot be read or tampered with in transit.

### Life Analogy
HTTP is sending a postcard. HTTPS is sending a locked safe where only the sender and receiver have the key.

### Key Points
- Encrypted via TLS.
- Secures data in transit.
