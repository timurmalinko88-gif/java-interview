Which method of the Runnable interface must be implemented to define thread logic?

A. start()
B. execute()
C. run()
D. call()

---ANSWER---
**Correct answer: C (run)**

### Key Points
- The `Runnable` interface contains a single abstract method `run()`.
- The `start()` method belongs to the `Thread` class and is used to actually launch a new system thread.
