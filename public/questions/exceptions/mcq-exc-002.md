С какой версии Java доступна конструкция try-with-resources?

A. Java 6
B. Java 7
C. Java 8
D. Java 9

---ANSWER---
**Правильный ответ: B (Java 7)**

### Key Points
- try-with-resources автоматизирует вызов метода `close()` у ресурсов, реализующих интерфейс `AutoCloseable` или `Closeable`.
