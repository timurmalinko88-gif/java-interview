Какой Scope у бинов в Spring установлен по умолчанию?

A. Prototype
B. Request
C. Session
D. Singleton

---ANSWER---
**Правильный ответ: D (Singleton)**

### Key Points
- По умолчанию контейнер Spring создает ровно один экземпляр бина (Singleton).
- Prototype создает новый экземпляр при каждом запросе из контекста.
